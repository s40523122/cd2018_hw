Nutcracker 零件繪製過程

Date 週二 10 十月 2017 By 40523122 Tags steps / samples

紀錄第四周回家練習之 Nutcracker 八個零件的繪製過程。
可先參考 W4課後整理 的課程筆記後，再觀看以下內容會比較容易上手。
若要組裝結合可參考下一篇: Nutcracker 零件組裝過程 的內容。

    • Base 繪製過程


    1. 照著工程圖所標示之尺寸於前基準面繪出，如下圖。
    # 可用建構線輔助矩形之中心定位。



    2. 填料長出 0.375 後便完成。


    Handle 繪製過程


    1. 照著工程圖所標示之尺寸於前基準面繪出，如下圖。



    2. 伸長填料 7.5 。



    3. 選擇如下圖中之兩線一點，建立一個新草圖 (Shift + w )。



    4. 於新草圖上繪出如工程圖所表達。



    5. 伸長除料 5.0 。



    6. 選擇如下圖中之兩線一點，再建立一個新草圖 (Shift + w )。



    7. 於新草圖上繪出如工程圖所表達。



    8. 伸長除料。
    # 選擇 difference 除料。



    9.選擇如下圖中一點一面，重合 ( O ) 後便完成。


    Adjuster 繪製過程


    1. 照著工程圖所標示之尺寸於前基準面繪出，如下圖，後點選圖中中線為旋轉軸，旋轉填料。
    # 以圖中中線為對稱軸，畫出建構線輔助尺寸之標示。



    2. 選擇如下圖中之兩線一點，建立一個新草圖 (Shift + w )。



    3. 於新草圖上繪出如工程圖所表達。



    4. 選擇半圓之底線旋轉除料後便完成。
    # 選擇 difference 除料。



    5. 若如下圖中產生紅色之干涉現象，可勾選 force NURBS surfaces to triangle mesh 解決。


    Piston 繪製過程


    1. 照著工程圖所標示之尺寸於前基準面繪出，如下圖，後點選圖中紅線為旋轉軸，旋轉填料。
    # 以圖中紅線為對稱軸，畫出建構線輔助尺寸之標示。



    2. 選擇如下圖中之點，於端面建立新草圖 (Shift + w )。



    3. 於新草圖上繪出如工程圖所表達。



    4. 伸長除料 1.0 。



    5. 選擇如下圖中之兩線一點，再建立一個新草圖 (Shift + w )。



    6. 於新草圖上繪出如工程圖所表達。



    7. 伸長除料，選擇如下圖中一點一面，重合 ( O )。



    8. 選擇如下圖中之兩線一點，再建立一個新草圖 (Shift + w )。



    9. 於新草圖上繪出如工程圖所表達，後如同 Adjuster 繪製過程之步驟4、5，便完成。


    Cylinder 繪製過程


    1. 照著工程圖所標示之尺寸於前基準面繪出，如下圖。



    2. 填料長出 3.0 。



    3. 選擇如下圖中之兩線一點，建立新草圖 (Shift + w )。



    4. 於新草圖上繪出如工程圖所表達。



    5. 除料伸長 0.5 後便完成。


    Hinge 繪製過程


    1. 照著工程圖所標示之尺寸於前基準面繪出，如下圖。



    2. 填料長出 0.75 。



    3. 選擇如下圖中之兩線一點，建立新草圖 (Shift + w )。



    4. 於新草圖上繪出如工程圖所表達。
    # 使用建構線輔助圓形之中心定位。



    5. 伸長除料，選擇如下圖中一點一面，重合 ( O )。



    6. 選擇如下圖中之兩線一點，再建立一個新草圖 (Shift + w )。



    7. 於新草圖上繪出如工程圖所表達。



    8. 伸長除料 0.5 後便完成。


    Adjuster_base 繪製過程


    1. 照著工程圖所標示之尺寸於前基準面繪出，如下圖。



    2. 伸長填料1.0。



    3. 選擇如下圖中之兩線一點，建立一個新草圖 (Shift + w )。



    4. 於新草圖上繪出如工程圖所表達。
    # 以圖中紅線為對稱軸，使兩側之圓對稱。



    5. 除料伸長 0.25 後便完成。


    Connection 繪製過程


    1. 照著工程圖所標示之尺寸於前基準面繪出，如下圖。



    2. 伸長填料 0.25 後便完成。
