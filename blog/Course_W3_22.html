W3課後整理

Date 週四 28 九月 2017 By 40523122 Tags notes / tips

本堂課是學期第三堂，也是前三周適應期的最後一堂，過了這堂課就要開始這門課的重點了，今天的內容為將前兩周未完成事項完成，並說明接下來的課程所需使用的 SolveSpace 和 Vrep 這兩套軟體，及調整好心態來準備、面對接下來會遇到的問題與工作。

課程目標

　　將前兩堂課未完成的內容完成，確定個人倉儲首頁有連結能點選進入個人 Blog ，確定能使用 leo editor 編輯網誌並回推遠端，確定每周回家練習並將心得打成網誌更新，學習 SolveSpace 基礎繪圖技巧，如何配合 SolveSpace 和 Vrep 將零件組成機構以供接下來的課程使用。

課程筆記

　　首先練習依照 Nutcracker 連結中的工程圖， 將零件以 SolveSpace 畫出，試著嘗試使用 SolveSpace 內的各種功能，並可參考 SolveSpace 說明 或草圖指引內的說明，並且練習利用 組立指引 練習組立，一步步了解程式的使用方式。
畫完的零件可用 File -> Export Triangle Mesh 轉為 .stl 檔，之後將 .stl 檔丟到 vrep 中做進階之動作分析。

SolveSpace 小技巧

1. W -> 將畫面正視於最近平面
2. 先選兩點(一點一線)後 D -> 標尺寸 # 不能選取兩線
3. 先選取兩線一點後 shift + w -> 於兩線一點所建構之面建立草圖
4. 先選取一點一線後 M -> 將選取之點至於線段中點
5. 選一線後 G -> 將選取之線轉為建構線 (綠線)
6. tab -> 將 Property Browser 關閉 (也就是除了主視窗外旁邊的小視窗)
7. 選取一點後 Ctrl + Shift + T -> 將接下來選取之點移動路徑畫出
