W5課後整理

Date 週四 12 十月 2017 By 40523122 Tags notes / steps / samples

今天是第五堂課，學習 SolveSpace 也已經三禮拜了，慢慢地比較熟悉軟體操作了。上堂課完成了使用 SolveSpace 繪製 Nutcracker 的八個零件，並使用限制條件組合成可做動之機構。今天的主要內容為將已繪製完成的 .stl 檔顯示在網誌中，使之可以做簡單的 3D 檢視。

課程目標

　　編輯近端檔案，使遠端網誌產生一視窗，能顯示完成之 .stl 零件檔。

成果展示

操作步驟

1. 在近端倉儲中建立一個名為 name 的資料夾
2. 開啟 leo editor 編輯器
3. 建立一節點 ( Ctrl + i )，名稱為 @path ./../name
4. 在 @path ./../name 下建立兩分支節點 ( Ctrl + i -> Ctrl + r )，分別為 @edit viewstl.html 和 @edit jsc3d.js
5. @edit viewstl.html 內容為: viewstl.html (進入後全選複製)
6. @edit jsc3d.js 內容為: jsc3d.js (進入後全選複製)

7. 接下來只要在網誌內容打上 
<iframe src="./../name/viewstl.html?src=stl檔案位置" width="870" height="900"></iframe>
即可於網誌上顯示。
